1. log in:
log in as admin using these credentials:
username: admin
password: admin

sign up:
you can also make an account using the sign up button under the user icon
then login with those credentials instead

2. shop:
Click the "Shop Now" button.
You will be sent to the products page.

As admin, you should be able to add a product using the form at the bottom of the page.
As admin, you should also be able to update and delete product information.

3. purchase:
Click on the "Add to Cart" button on any product to add it to the cart. 
You will be sent to the cart page.
Go back to the products page to add more to the cart.
You can click on the "Remove Item from Cart" button to remove an unwanted item.
You can click on the "Confirm Purchase" button to confirm your purchase.
